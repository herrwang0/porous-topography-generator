# Porous Topography Generator (ptopo)
**Porous Topography Generator** is a Python package for generating sub-grid topography from high-resolution data based on the algorithm in

> Adcroft, A. (2013). *Representation of topography by porous barriers and objective
> interpolation of topographic data*. Ocean Modelling, 67, 13â€“27.

## Installation

```bash
git clone https://github.com/herrwang0/porous-topography-generator.git
cd porous-topography-generator

pip install -e .

```

## CLI
> ptopo regrid
